# venvy - менеджер виртуальных окружений

Команды:

venvy
- создает окружение default (если нет)
- заходит в него

venvy use <имя>
- создает окружение с указанным именем (если нет)
- заходит в него

venvy list
- показывает список всех окружений

venvy remove <имя>
- удаляет окружение

venvy run <имя> <скрипт>
- запускает скрипт в указанном окружении

Переменные:

VENVY_HOME - куда складывать окружения (по умолчанию ~/.venvy)

Окружения хранятся в ~/.venvy/
